### Executando

### 1. Clone o repositório

```bash
git@github.com:tavareshenrique/podcastr-nlw.git
```

### 2. Instale as dependencias

```bash
npm install
```

### 3. Inicie o servidor através do JSON Sever

```bash
json-server server.json -w -p 3333
```

### 5. Inicie o Projeto

```bash
npm run start
```
